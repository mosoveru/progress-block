# Progress Demonstration

Демонстрация работы Progress компонента в рамках тестового задания на стажировку в Ozon.

## ProgressComponent API

Чтобы переиспользовать компонент на странице, достаточно подключить стили `progress_styles.css` с помощью тега `<link>` к странице и вызвать метод `mount()` у экземпляра компонента, где в качестве аргумента предоставить контейнер, в котором будет отрисован сам компонент.

Instance Methods

- `mount(container)` - метод монтирует компонент в предоставленный пользователем контейнер
- `setArcValue(value)` - метод устанавливает значение Progress, который визуально заполняется в браузере в виде дуги. В качестве аргумента принимает число. 0 возвращается, если в метод было передано некорректное значение или отрицательное значение. 100 возвращается, если было передано значение больше 100. Во всех остальных случаях возвращается `value`.
- `setAnimate(isAnimated)` - метод включает/отключает анимацию вращения дуги компонента. Принимает в качестве аргумента Boolean значение. Анимация не будет работать, если к странице не подключены стили `progress_styles.css`.
- `setHidden(isHidden)` - метод скрывает/показывает компонент на странице. Принимает в качестве аргумент Boolean значение.
- `destroy()` - метод удаляет компонент из контейнера, к которому он был примонтирован.
- `getState()` - метод возвращает текущее состояние компонента. Возвращаемое значение - объект со свойствами value, animated, hidden, которые содержат значения выставленного Progress, состояние анимации и отображения элемента на странице соответственно.

Constructor

Компонент при инстанцировании может принимать объект с опциями, с помощью которых можно настроить его поведение. Опции объекта:

- `size` - размер компонента в пикселях. По умолчанию `160`.
- `minValue` - задаёт минимальное значение, которое компонент может принять в качестве состояния шкалы прогреса. По умолчанию 0;
- `maxValue` - задаёт максимальное значение, которое компонент может принять в качестве состояния шкалы прогреса. По умолчанию 100;
- `stroke` - ширина кольца и самой дуги компонента в пикселях. По умолчанию `12`.
- `trackColor` - цвет кольца в виде строки со значением цвета в HEX-формате. По умолчанию `#EEF3F6`.
- `arcColor` - цвет дуги в виде строки со значением цвета в HEX-формате. По умолчанию `#015DFE`.
- `arcTransitionDelay` - установить плавность перехода состояния компонента. Принимает строку со значением, аналогичным тому, что принимает CSS свойство transition, например, `1s` или `2s`. По умолчанию `1s`.
